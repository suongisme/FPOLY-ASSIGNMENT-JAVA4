# Website Xem Video

## Mô tả:
Cho phép người dùng đăng tải video, quản lý video cá nhân.

## Chức năng
- Đăng ký tài khoản
- Đăng nhập
- Quản lý tài khoản cá nhân
- Thêm, sửa,xóa video
- Like/Dislike video

## Công nghệ
- Servlet, Hibernate, Mysql
